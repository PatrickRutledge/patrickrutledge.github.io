How It Works

The app allows users to make simple charts by pasting in comma-separated series data. The user can choose the unit of measure for time and the scale of measure, set a goal, and define minimum and maximum levels. The chart automatically produces a mean line based on the series data.

Chart showing color-coded lines for different thresholds and measurements

Understanding the Chart Elements:

  • Upper & Lower Spec Limits (Blue lines) - Define the acceptable range for your data points
  • Goal Line (Green line) - A target threshold that users can set for desired performance
  • Mean Line (Red line) - Automatically calculated average of all data points to show actual performance

This approach goes beyond traditional run charts by combining multiple types of reference lines. For production scenarios:

  • The lower specification limit can represent the breakeven point
  • The upper specification limit indicates maximum possible production
  • The green goal line shows the target production level
  • The red mean line reveals actual performance against these thresholds

Run Chart Visualization can be used for:

  • Statistical analysis using upper and lower specification limits
  • Production and throughput monitoring (pounds or tons of aggregate, number of widgets)
  • Time series analysis across various intervals (minutes, hours, days, months, or years)
  • Goal setting and performance tracking against established targets

With its simple interface, the app was designed for use on the shop floor from laptops or Windows mobile devices. Charts can be opened in a new tab, saved to PDF, or shared via email, text, or other methods - making it easy for production floor personnel to illustrate activity and take quick measurements.
